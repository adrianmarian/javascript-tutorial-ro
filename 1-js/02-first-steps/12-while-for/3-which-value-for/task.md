# Ce valori sunt afișate de către bucla "for"?

Pentru fiecare buclă scrie ce valori vor fi afișate. Apoi compară cu răspunsul.

Ambele bucle `alertează` aceleași valori sau nu?

1. Forma postfixată:

    ```js
    for (let i = 0; i < 5; i++) alert( i );
    ```
2. Forma prefixată:

    ```js
    for (let i = 0; i < 5; ++i) alert( i );
    ```
