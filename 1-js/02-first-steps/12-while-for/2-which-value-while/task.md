# Ce valori afișează bucla while?

Pentru fiecare iterație a buclei, scrie ce valoare va afișa și compar-o cu soluția.

Ambele bucle `alertează` aceleași valori, sau nu?

1. Forma prefixată `++i`:

    ```js
    let i = 0;
    while (++i < 5) alert( i );
    ```
2. Forma postfixată `i++`

    ```js
    let i = 0;
    while (i++ < 5) alert( i );
    ```
